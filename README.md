# How to use?

Install
```bash
npm i
```
Development
```bash
npm start
npm test
npm run tdd
```

Production
```bash
npm run build-prod
npm run serve
```

For further information take a look at [this blog post](http://blog.mgechev.com/2016/06/26/tree-shaking-angular2-production-build-rollup-javascript/).

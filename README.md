# Angular2 Simple Build

### Build
`npm install`

### Development
- `npm start` (live server)
- `npm test` (test once)
- `npm run tdd` (tdd)

### Production
- `npm run build-prod` (prepare bundle)
- `npm run serve` (serve content)

### Reports
- Tests: `karma/report/tests/index.html`
- Coverage: `karma/report/coverage/index.html`
